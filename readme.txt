ABOUT THIS PROJECT
------------------------
This project is a game involving dungeons and dragons! The user will start with a party
of 5 players and will navigate through the dungeon and fight monsters and find jewelry and keys!
The user will have to move logically and to make sure the sorcerer's anger never reaches 100.
The user will have to keep the players healthy and make sure to keep protection throughout the 
dungeon and prepare for the sorcerer!
